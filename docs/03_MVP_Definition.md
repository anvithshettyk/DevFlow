# Minimum Viable Product (MVP)

## Objective

The objective of Version 1.0 is to provide a functional project management platform that allows users to create projects, manage tasks, collaborate with team members, and track project progress.

## Included Features

### Authentication
- Register
- Login
- Logout

### Dashboard
- View Projects
- View Assigned Tasks

### Project Management
- Create Project
- Edit Project
- Delete Project

### Task Management
- Create Task
- Assign Task
- Update Task Status
- Delete Task

### Kanban Board
- To Do
- In Progress
- Done

### Team Management
- Invite Members

### User Profile
- Update Profile
